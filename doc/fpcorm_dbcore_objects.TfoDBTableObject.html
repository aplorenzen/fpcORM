function OpenConnection: Boolean;

The OpenConnection function provides the trivial means of opening the database connection to the decendant types.

Returns

True if the connection was opened successfully, False if connection could not be established.

function CloseConnection(aConsumeException: Boolean = False): Boolean;

The CloseConnection function closes the local connection. Only if it is actually open and if the decending object also was the initiator of the connection. Making this distinction allows decendants to use the CloseConnection function anywhere in decendant types, where they may want to close the database connection, and only close it if that decendant object instance was the object that opened it. .

Parameters
aConsumeException
indicates to the function, if any exception caused should be silenced, or re-raised to the caller.
Returns

False if unsuccessful and True if closed successfully

procedure AddTableField(aTableField: TfoDBAbstractTableField);

AddTableField is intended to be called from the constructors of the fields that are owned by the decending object from this class. Upon creation, a new field (decending from TfoDBAbstractTableField) is passed it's owning table as a constructor parameter. The field should in turn then call it's owner's AddTableField procedure, to add itself to it's table's field collection. So that the table object has a complete list of all the fields that are contained in it.
