Class TfoDBAbstractTableField

DescriptionHierarchyFieldsMethodsProperties

Unit

fpcorm_dbcore_objects

Declaration

type TfoDBAbstractTableField = class(TfoObserverSubject)

Description

This type represents a field in a table, a specific field in a specific row, but without it's value at this abstract level. All the other properties of the field are implemented at this level though.
